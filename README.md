# BlockChainPython
Making block chain and connecting multiple devices with it. 

# P2P Flow
The connections will be handled by a thread and will handle the reading and sending of data on all connections.

## Service Node
This is a unique node that will be only for storing the ip addresses of the P2P network along with data of the current blockchain. This node will be for getting information only, not processing the blockchain.
